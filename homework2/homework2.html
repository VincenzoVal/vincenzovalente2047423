What is a Dataset?

A dataset is a structured collection of data. It can be represented as a table with rows and columns,  where each row corresponds to an observation and each column represents a variable or feature.  Datasets are essential in data analysis, statistics, and machine learning, as they provide the foundation for discovering patterns, relationships, and trends.

For example, a dataset about people might include their age, height, and weight. This data can later be used to study distributions, correlations, and predictions.

What is a Distribution?

A distribution describes how the values of a variable are spread or arranged. It shows the frequency or probability  of each possible outcome. Distributions are fundamental in statistics, as they allow us to summarize and understand data behavior.

There are two main types of statistical distributions:

     
  • Discrete distribution – deals with countable outcomes (e.g., number of login attempts).
    •  
  • Continuous distribution – deals with outcomes that can take any value within a range (e.g., height, weight).

Univariate Distribution Example

A univariate distribution focuses on a single variable.  Below we compute the frequency distribution of Age from our dataset. 


-- Frequency distribution of Age
SELECT Age, COUNT(*) AS Frequency
FROM People
GROUP BY Age
ORDER BY Age;

The Caesar Cipher: An Application of Distributions

The Caesar cipher is one of the simplest and most widely known encryption techniques. It's a type of substitution cipher where each letter in the plaintext is shifted a certain number of places down or up the alphabet. For example, with a left shift of 3, 'D' would be replaced by 'A', 'E' would become 'B', and so on. The method is named after Julius Caesar, who used it in his private correspondence.

Decoding with Frequency Analysis

While simple, the Caesar cipher is easily broken using frequency analysis. In any given language, certain letters appear more frequently than others. In English, 'E' is the most common letter, followed by 'T', 'A', 'O', 'I', 'N', etc.

A Caesar cipher shifts the entire alphabet, so the letter frequency distribution of the ciphertext is just a shifted version of the plaintext's distribution. To find the key, you can:

     
  1. Calculate the frequency of each letter in the ciphertext.
    2.  
  2. Find the most frequent letter.
    3.  
  3. Assume this letter corresponds to the most common letter in the language (e.g., 'E' in English).
    4.  
  4. The difference between these two letters is the shift key.

Encryption Function


function encrypt(text, shift) {
    let result = "";
    for (let i = 0; i < text.length; i++) {
        let char = text[i];
        if (char.match(/[a-z]/i)) {
            let code = text.charCodeAt(i);
            // Uppercase letters
            if ((code >= 65) && (code <= 90)) {
                char = String.fromCharCode(((code - 65 + shift) % 26) + 65);
            }
            // Lowercase letters
            else if ((code >= 97) && (code <= 122)) {
                char = String.fromCharCode(((code - 97 + shift) % 26) + 97);
            }
        }
        result += char;
    }
    return result;
}

Frequency Analysis and Shift Detection


function findShift(ciphertext) {
    // English letter frequency order: E is most common (index 4)
    const englishMostCommonIndex = 4; // 'E'
    const frequencies = new Array(26).fill(0);

    for (let i = 0; i < ciphertext.length; i++) {
        let char = ciphertext[i];
        if (char.match(/[a-z]/i)) {
            let code = char.toUpperCase().charCodeAt(0);
            frequencies[code - 65]++;
        }
    }

    let maxFreq = -1;
    let mostFrequentIndex = -1;
    for (let i = 0; i < frequencies.length; i++) {
        if (frequencies[i] > maxFreq) {
            maxFreq = frequencies[i];
            mostFrequentIndex = i;
        }
    }

    if (mostFrequentIndex === -1) return 0; // No letters found

    // The shift is the difference between the most frequent letter's
    // index and 'E's index.
    const detectedKey = (mostFrequentIndex - englishMostCommonIndex + 26) % 26;
    return detectedKey;
}

Decryption Function

Decrypting is the same as encrypting with a shift in the opposite direction.


function decrypt(ciphertext, shift) {
    // Decryption is just encryption with the inverse shift
    return encrypt(ciphertext, 26 - shift);
}
